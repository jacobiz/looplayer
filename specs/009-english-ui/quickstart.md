# Quickstart: 英語 UI 対応 手動テストシナリオ

**Feature**: `009-english-ui`
**Date**: 2026-03-16

## 前提

```bash
python main.py
```

---

## シナリオ 1: 英語ロケールで起動（主要検証）

**テスト環境の準備**:
- Linux/macOS: `LANG=en_US.UTF-8 python main.py`
- Windows: システムロケールを English (United States) に変更してから起動

**確認項目**:
1. メニューバーが `File`, `Playback`, `View`, `Help` で表示される
2. 動画を開くと再生ボタンが `Play` → 一時停止で `Pause` になる
3. 音量ラベルが `Volume:` で表示される
4. ABループボタンが `AB Loop: OFF` / `AB Loop: ON` で表示される
5. ブックマークパネルのタイトルが `Bookmarks`、ボタンが `Sequential Play` で表示される
6. 存在しないファイルを開こうとするとエラーダイアログが英語で表示される

---

## シナリオ 2: 日本語ロケールで起動（リグレッション確認）

**テスト環境の準備**:
- Linux/macOS: `LANG=ja_JP.UTF-8 python main.py`
- Windows: システムロケールを日本語に変更してから起動

**確認項目**:
1. メニューバーが `ファイル`, `再生`, `表示`, `ヘルプ` で表示される（既存通り）
2. すべての UI テキストが日本語のまま
3. 機能動作に変化がないこと

---

## シナリオ 3: 日英以外のロケールで起動（フォールバック確認）

**テスト環境の準備**:
- `LANG=fr_FR.UTF-8 python main.py`

**確認項目**:
1. UI が英語で表示される（フランス語ではなく英語にフォールバック）
2. クラッシュや例外が発生しない

---

## 合格基準チェックリスト

- [ ] シナリオ 1: 目に見えるすべての UI テキストが英語（日本語テキスト混在 0 件）
- [ ] シナリオ 2: すべての UI テキストが従来どおり日本語
- [ ] シナリオ 3: 英語でフォールバック表示・クラッシュなし
- [ ] ユーザー入力テキスト（ブックマーク名・メモ）は翻訳されず入力通りに表示
