# Feature Specification: 英語 UI 対応

**Feature Branch**: `009-english-ui`
**Created**: 2026-03-16
**Status**: Draft
**Input**: User description: "英語対応"

## User Scenarios & Testing *(mandatory)*

### User Story 1 - OS 言語が英語のとき UI が英語で表示される (Priority: P1)

OS の表示言語が日本語以外（英語など）に設定されているユーザーがアプリを起動すると、メニュー・ボタン・ラベル・ダイアログ・エラーメッセージなどのすべての UI テキストが英語で表示される。現在はすべてのテキストが日本語で固定されており、日本語を理解しないユーザーはアプリを使用できない。

**Why this priority**: 英語対応の根幹であり、これが実現すれば日本語以外の OS を使うユーザーがアプリを自力で利用できるようになる。他のすべての UI 改善の前提となる。

**Independent Test**: 英語ロケールの環境でアプリを起動し、メニュー・ボタン・ダイアログ・ステータスバーのテキストがすべて英語で表示されることを確認することで独立してテスト可能。

**Acceptance Scenarios**:

1. **Given** OS 言語が英語に設定されている, **When** アプリを起動する, **Then** メニューバー（File, Playback, View, Help）が英語で表示される
2. **Given** OS 言語が英語に設定されている, **When** 動画を開く, **Then** 再生ボタン・停止ボタン・音量ラベルなどのコントロールが英語で表示される
3. **Given** OS 言語が英語に設定されている, **When** ファイルが見つからない場合, **Then** エラーダイアログが英語で表示される
4. **Given** OS 言語が日本語に設定されている, **When** アプリを起動する, **Then** すべての UI テキストが従来どおり日本語で表示される（既存挙動の維持）

---

### Edge Cases

- OS 言語が英語・日本語以外（フランス語・中国語など）の場合は英語にフォールバックする
- 動画ファイル名・ブックマーク名などのユーザー入力テキストは翻訳対象外（そのまま表示）
- ステータスバーの動的メッセージ（再生速度変更通知など）も UI 言語に従う

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: OS の表示言語が日本語の場合、UI テキストをすべて日本語で表示しなければならない（既存動作の維持）
- **FR-002**: OS の表示言語が日本語以外の場合、UI テキストをすべて英語で表示しなければならない
- **FR-003**: 英語対応が必要なテキスト範囲は以下を含む：メニューバー・サブメニュー・アクション名、再生/停止/音量などのコントロールラベルとツールチップ、ダイアログタイトルとメッセージ（エラー・確認・情報）、ステータスバーの通知メッセージ、ブックマークパネルの見出しとボタンラベル
- **FR-004**: ユーザーが入力したテキスト（ブックマーク名・動画ファイルパス・メモなど）は翻訳対象外とし、入力した言語のまま表示しなければならない
- **FR-005**: OS の表示言語が英語・日本語以外の場合は英語にフォールバックしなければならない
- **FR-006**: 言語設定はアプリ起動時に一度だけ決定し、起動後に変更されても再起動まで影響しない（再起動不要）

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 英語ロケール環境でアプリを起動したとき、目に見えるすべての UI テキストが英語で表示される
- **SC-002**: 日本語ロケール環境でアプリを起動したとき、すべての UI テキストが従来どおり日本語で表示され、リグレッションが発生しない
- **SC-003**: 英語・日本語以外のロケールで起動したとき、UI が英語で表示される（クラッシュなし）
- **SC-004**: 翻訳対象のすべてのメニュー・ボタン・ダイアログ・エラーメッセージにおいて、英語テキストの欠落（日本語テキストの混在）が 0 件

## Clarifications

### Session 2026-03-16

- Q: 言語切替の方法（OS 自動検出のみか、アプリ内手動切替か） → A: OS ロケール自動検出のみ（Option A）。手動切替なし、設定 UI 不要。

## Assumptions

- 対応言語は日本語と英語の 2 言語のみ（他言語への拡張は本仕様の範囲外）
- 言語切替は OS ロケールの自動検出のみ（手動切替なし、アプリ設定 UI 不要）
- キーボードショートカットのキー文字（`[`, `]` など）は言語に依存しないため翻訳不要
- ショートカット一覧ダイアログの説明文も翻訳対象とする
