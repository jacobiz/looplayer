# Quickstart: ビデオプレイヤーコア機能

**Date**: 2026-03-15

## 前提条件

- Python 3.12+
- VLC メディアプレイヤーがシステムにインストール済み
- WSL2 環境の場合: WSLg (DISPLAY=:0) が有効

## セットアップ

```bash
pip install -r requirements.txt
pip install pytest pytest-qt  # テスト用
```

## アプリ起動

```bash
python main.py
```

## 動作確認チェックリスト

### US1: 基本再生

- [ ] 「開く」ボタンで動画ファイルを選択 → 自動再生が始まる
- [ ] 再生中に「一時停止」ボタン → 停止、ボタンが「再生」に変わる
- [ ] 「再生」ボタン → 再開
- [ ] 「停止」ボタン → 先頭に戻り、時刻が「00:00 / 00:00」になる
- [ ] シークバーをドラッグ → 再生位置が変わる
- [ ] 時刻表示が再生進行に合わせて更新される

### US2: ABループ

- [ ] 再生中に「A点セット」→ 「A: MM:SS」が表示される
- [ ] 少し進めて「B点セット」→ 「B: MM:SS」が表示される
- [ ] 「ABループ: OFF」ボタン → 「ABループ: ON」に変わる
- [ ] B点に達すると自動的にA点に戻る
- [ ] 「ABリセット」→ 「A: --  B: --」になりループがOFFになる

### エラーハンドリング

- [ ] 破損したファイルを開く → エラーダイアログが表示され、直前の状態が維持される

## テスト実行

```bash
# ユニットテスト（ヘッドレス環境でも動作）
pytest tests/unit/ -v

# 統合テスト（GUI 環境必須）
pytest tests/integration/ -v

# 全テスト
pytest tests/ -v
```
