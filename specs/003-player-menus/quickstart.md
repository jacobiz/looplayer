# Quickstart: プレイヤーメニュー基本機能

**Date**: 2026-03-15

## 実装フロー（3ステップ検証）

### Step 1: US1 — ファイルメニューとショートカット

```bash
python main.py
```

1. メニューバーに「ファイル」「再生」「表示」が表示されることを確認
2. 「ファイル > 開く」をクリック → ファイル選択ダイアログが開く
3. `Ctrl+O` を押す → ファイル選択ダイアログが開く
4. 動画を開いた後 `Space` → 一時停止、もう一度 `Space` → 再生再開
5. `Ctrl+Q` → アプリ終了

---

### Step 2: US2 — 再生コントロールメニューとシーク

1. 動画を開く
2. 音量スライダーが表示されていることを確認（コントロールバーに "80%" ラベル）
3. `↑` キー → 音量が上昇し、ラベルが "90%" に変わる
4. `M` キー → ミュート（ラベルが "0%"）、再度 `M` → 元の音量に復元
5. `→` キー → 5秒前進、`←` キー → 5秒後退
6. 「再生 > 再生速度 > 1.5倍」 → 動画が速くなる（選択肢にチェックマーク）
7. 別のファイルを開く → 速度が標準（1.0倍）に自動リセットされる

---

### Step 3: US3 — 表示メニューとフルスクリーン

1. `F` キー → フルスクリーン（コントロール・ブックマークパネルが消え映像のみ）
2. マウスカーソルを画面最上部に移動 → メニューバーが 2 秒間表示される
3. `Esc` キー → 通常ウィンドウに戻る（コントロール群が再表示）
4. 「表示 > 常に最前面に表示」 → メニューにチェックマーク、他ウィンドウの前面に固定
5. 再度「表示 > 常に最前面に表示」 → チェックが外れ、通常の重ね順に戻る

---

```bash
pytest tests/unit/test_volume_controls.py -v    # 音量・ミュート状態テスト
pytest tests/unit/test_playback_speed.py -v     # 再生速度状態テスト
pytest tests/integration/test_menus.py -v       # メニュー・ショートカット統合テスト
pytest tests/integration/test_fullscreen.py -v  # フルスクリーン統合テスト
pytest tests/ -v                                # 全テスト（回帰確認）
```

## キーボードショートカット一覧

| キー | 操作 |
|------|------|
| `Ctrl+O` | ファイルを開く |
| `Ctrl+Q` | アプリを終了 |
| `Space` | 再生/一時停止 |
| `→` | 5秒前進 |
| `←` | 5秒後退 |
| `↑` | 音量 +10% |
| `↓` | 音量 -10% |
| `M` | ミュート切り替え |
| `F` | フルスクリーン切り替え |
| `Esc` | フルスクリーン解除 |
