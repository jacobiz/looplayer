## 各仕様書のファイル構成

各ディレクトリには以下のファイルが含まれます。

| ファイル | 内容 |
|---------|------|
| `spec.md` | ユーザーストーリー・機能要件・受け入れ条件 |
| `plan.md` | 技術スタック・実装設計・変更ファイル一覧 |
| `research.md` | 技術調査・決定事項の根拠 |
| `quickstart.md` | 動作確認シナリオ |
| `tasks.md` | 実装タスク一覧（チェックボックス形式） |
| `data-model.md` | データモデル定義（一部のみ） |
| `checklists/` | 仕様品質チェックリスト（一部のみ） |
